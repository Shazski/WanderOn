## Installation

1. **Clone the repository**

   ```sh
   git clone https://github.com/your-username/vite-react-node-project.git
   cd vite-react-node-project
2. **Install dependencies for the frontend**   
    cd client
    npm install
3. **Install dependencies for the backend**
    cd ../server
    npm install
